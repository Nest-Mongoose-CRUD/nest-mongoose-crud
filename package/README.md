# Handler Factory

A reusable NestJS handler factory for building CRUD operations with Mongoose.

## Installation

```bash
npm install @yourorg/handler-factory
# or
yarn add @yourorg/handler-factory
```

## Usage

```typescript
import { createHandlerFactory } from '@yourorg/handler-factory';

// Use in your NestJS services
const handler = createHandlerFactory();
```

## Features

- 🎯 Simple factory pattern for creating handlers
- 📦 Works with NestJS and Mongoose
- 🧪 Fully typed with TypeScript
- 🚀 Production-ready
